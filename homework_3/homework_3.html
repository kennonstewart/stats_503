Question 1: ISLP Chapter 4, Conceptual Exercise #1

Problem Statement:

Using a little bit of algebra, prove the following equation: $$ p(x) = \frac{\exp(\beta_{0} + \beta_{1}X)}{1+\exp(\beta_{0} + \beta_{1}X)} $$ Show that this simplifies to: $$ \log\left[\frac{p(X)}{1-p(X)}\right] = \beta_{0}+\beta_{1}X $$

Solution:

Using logarithm properties:

$$ \log\left[\frac{p(X)}{1-p(X)}\right] = \log[p(X)] - \log[1-p(X)] $$

Substituting the logistic function:

$$ \log\left[\frac{\exp(\beta_{0} + \beta_{1}X)}{1+\exp(\beta_{0} + \beta_{1}X)}\right] - \log\left[1 - \frac{\exp(\beta_{0} + \beta_{1}X)}{1+\exp(\beta_{0} + \beta_{1}X)}\right] $$

Simplifying:

$$ \log[\exp(\beta_{0}+\beta_{1}X)] - \log[1+\exp(\beta_{0}+\beta_{1}X)] - \log[1] + \log[1+\exp(\beta_{0}+\beta_{1}X)] $$

$$ = \beta_{0}+\beta_{1}X $$

Problem Statement:

We analyze how increasing the number of features affects prediction accuracy.

Part A:

If (X) is uniformly distributed on ([0,1]), estimate the fraction of available observations used for prediction when considering only points within 10% of the range of (X).

Solution:

Since predictors are uniformly distributed, the range ([X-0.05, X+0.05]) contains 10% of the data.

Part B:

Now suppose observations have two features, (X_1) and (X_2), uniformly distributed on ([0,1] \times [0,1]). What fraction of available observations will be used?

Solution:

The probability of selecting an observation in both dimensions:

$$ (0.1) \times (0.1) = 0.01 $$

Thus, 1% of the observations are used.

Part C:

Extend the result to ( p = 100 ) features.

Solution:

For ( p ) dimensions, the fraction is:

$$ (0.1)^{100} = 1 \times 10^{-99} $$

This is an extremely small proportion of the dataset.

Part D:

Discuss the implications of increasing dimensions on predictive performance.

Solution:

With increasing dimensions, the fraction of data used becomes negligible, making local methods ineffective. This is known as the curse of dimensionality.

Part E:

Derive the expression for the side length of an ( n )-dimensional hypercube.

Solution:

The side length ( s ) of an ( n )-dimensional hypercube that captures a fraction ( f ) of observations is:

$$ s = (0.1)^{\frac{1}{n-1}} $$

This ensures the volume integrates to 1.

Question 3: ISLP Chapter 4, Conceptual Exercise #9

Problem Statement:

Compute the probability and odds of credit card default based on given odds.

Part A:

Given odds of default = 0.37, compute ( P(X) ).

Solution:

Using:

$$ \text{Odds}(X) = \frac{p(X)}{1-p(X)} $$

We solve:

$$ 0.37 = \frac{p(X)}{1-p(X)} $$

Rearrange:

$$ p(X) = \frac{0.37}{1.37} \approx 0.27 $$

Thus, 27% probability of default.

Part B:

Given 16% probability, compute odds.

Solution:

$$ \text{Odds} = \frac{0.16}{1-0.16} = 0.19 $$

Thus, odds of 0.19.

Part A:

Predict probability when X1 = 5, X2 = 3.5:

$$ p(Y=1) = \frac{\exp(-4 + 0.05(5) + 3.5)}{1 + \exp(-4 + 0.05(5) + 3.5)} $$

Part B:

Compute odds:

$$ \text{Odds} = \frac{p(X)}{1 - p(X)} $$ $$ \text{Odds} = \frac{ exp(-0.25) }{ 1+exp(-0.25) } $$ $$ \text{Odds} = exp(-.25) $$

Part C:

Solve for $X_1$ when $p(Y=1) = 0.5$:

Using:

$$ \log \frac{p(Y=1)}{1 - p(Y=1)} = -4 + 0.05X_1 + X_2 $$

Since $p(Y=1) = 0.5$, we set:

$$ 0.5 = \frac{\exp(-4 + 0.05X_1 + X_2)}{1 + \exp(-4 + 0.05X_1 + X_2)} $$

Taking the logarithm:

$$ \log 1 = 0 $$

$$ -4 + 0.05X_1 + X_2 = 0 $$

Solving for $X_1$:

$$ X_1 = \frac{4 - X_2}{0.05} $$

This gives the threshold for $X_1$ when the probability of success is exactly 50%.

# modules.py
import numpy as np


def hard_classification(threshold: int, probabilities:np.ndarray) -> np.array:
    results = [0] * len(probabilities)
    for ind in range(len(probabilities)):
        if probabilities[ind][1] >= threshold:
            results[ind] = 1
    return results

def confusion_matrix(y_true:np.array, y_predicted:np.array) -> np.ndarray:
    FP = TP = FN = TN = 0 # initialize the rate values to 0
    y_true = [1 if prediction == "Yes" else 0 for prediction in y_true]
    for prediction, truth in zip(y_predicted, y_true):
        if (prediction == 1) and (truth == 1): # true positive
            TP += 1
        elif (prediction == 1) and (truth == 0): # false positive
            FP += 1
        elif (prediction == 0) and (truth == 0): # true negative
            TN += 1
        else: # false negative
            FN += 1
    conf_matrix = np.array([[TN, FN], [FP, TP]])

    return conf_matrix

def rate_calculations(test_set:np.ndarray, classifications:np.array) -> tuple:
    # calculate the confusion matrix
    matrix = confusion_matrix(test_set, classifications)

    # assign the TP, FP, TN, FN rates
    TN = int(matrix[0][0])
    FN = int(matrix[0][1])
    FP = int(matrix[1][0])
    TP = int(matrix[1][1])

    # status check
    # print(f"TP: {TP}")
    # print(f"FP: {FP}")
    # print(f"TN: {TN}")
    # print(f"FN: {FN}")

    # calculate TPR
    TPR = TP / max((TP + FN), 1)
    # calculate FPR
    FPR = FP / max((FP + TN), 1)
    # calculate TNR
    TNR = TN / max((TN + FP), 1)
    # calculate FNR
    FNR = FN / max((FN + TN), 1)

    return (TPR, FPR, TNR, FNR)

def calculate_nll(probabilities: np.array) -> float:
    sum = 0
    for no_prob, yes_prob in probabilities:
        sum -= np.log(max(yes_prob, no_prob))
    return(float(sum))

def simulate_thresholds(prediction_probabilities: np.ndarray, y_test: np.array) -> list:   
    # initialize the range of thresholds
    thresholds = np.linspace(0, 1, 101)
    results = list()
    for threshold in thresholds:
        # compute the classifications
        classifications = hard_classification(threshold, prediction_probabilities)

        # calculate the prediction rate results
        TPR, FPR, TNR, FNR = rate_calculations(y_test, classifications)

        # add to the results
        results.append((TPR, FPR, TNR, FNR))
        
    # return the results
    return (results)
